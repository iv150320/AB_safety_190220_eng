# AB_safety_190220_eng: Interactive E-Learning Module 🎓🚀

This repository hosts the web-published output of an **Articulate Storyline 360** e-learning module, titled `AB_safety_190220_English`. Designed to deliver comprehensive safety training, this interactive course leverages modern web technologies (HTML5, CSS3, and JavaScript) to provide a rich, engaging learning experience directly within a standard web browser. It serves as a standalone educational resource for individuals seeking foundational knowledge or refresher training on specific safety protocols.

## 🎯 Overview

`AB_safety_190220_eng` is more than just a standard web application; it's a meticulously crafted digital learning experience. It represents the final, deployed version of an e-learning course developed using Articulate Storyline 360, a leading authoring tool in the e-learning industry. The content focuses on "safety," as indicated by the project name, and is presented in English.

The project structure reflects Storyline's efficient publishing process, generating a highly optimized, responsive web application ready for deployment on any web server or for direct local viewing.

## 🚀 Key Features

*   **Comprehensive Safety Curriculum**: Delivers targeted educational content focused on essential safety principles and practices.
*   **Interactive Learning Experience**: Engages learners through interactive slides, clickable elements, and potentially quizzes or knowledge checks (inherent to Storyline courses).
*   **Multimedia Integration**: Incorporates various media types (images, potentially audio/video, animations) to enhance understanding and engagement.
*   **Responsive Web Design**: Optimized for seamless viewing and interaction across a wide range of devices, including desktops, tablets, and mobile phones, ensuring accessibility for all learners.
*   **Standalone Web Deployment**: The entire course can be hosted on a standard web server or run locally, requiring no specialized software beyond a web browser.
*   **Modular Content Structure**: Content is organized into a clear, navigable structure, allowing learners to progress through topics systematically.
*   **Basic Analytics Integration**: Includes components (`analytics-frame.html`, `user.js`) that suggest capabilities for basic tracking of learner progress or interactions within a learning environment.

## 🛠 Technology Stack

This project is primarily the *output* of a sophisticated authoring tool, rather than being built from scratch with these technologies.

*   **Primary Authoring Tool**:
    *   **Articulate Storyline 360**: The industry-standard software used to design, develop, and publish the interactive e-learning course.
*   **Output Technologies (Generated by Storyline)**:
    *   **HTML5**: Provides the structural foundation of the web application, defining the layout and elements of each course slide.
    *   **CSS3**: Styles the course interface, ensuring a consistent and visually appealing presentation across all slides and interactive elements.
        *   *Generated Stylesheets*: `html5/data/css/output.min.css`, `html5/lib/stylesheets/main.min.css`.
    *   **JavaScript (Vanilla JS)**: Powers the interactivity, navigation, multimedia control, and overall dynamic behavior of the e-learning module. This includes Storyline's proprietary runtime and logic.
        *   *Core Runtime Files*: `html5/lib/scripts/app.min.js`, `story_content/story.js`, `html5/data/js/data.js`, `html5/data/js/frame.js`, `html5/data/js/paths.js`, and numerous other minified and uniquely named `.js` files within `html5/data/js/`.
    *   **XML**: Used for internal configuration and metadata, such as `meta.xml` and `story_content/frame.xml`.
*   **Assets**:
    *   `JPEG` and `PNG` image files for graphics, icons, and multimedia elements, many of which are optimized for mobile display within the `mobile/` directory.
    *   `WOFF` web fonts (`html5/lib/stylesheets/mobile-fonts/`) for consistent typography across devices.

## 🏗 Architecture / Workflow

The architecture of `AB_safety_190220_eng` follows a typical e-learning development and deployment pattern, centered around Articulate Storyline 360 as the authoring tool and standard web technologies for delivery.

1.  **Authoring (Developer Side)**: An e-learning developer uses **Articulate Storyline 360** to create the course content, slides, interactions, quizzes, and multimedia elements.
2.  **Publishing**: Storyline 360 then publishes this content to a "Web" or "HTML5" output, generating all the necessary HTML, CSS, JavaScript, and asset files. This process includes optimizing assets for various devices.
3.  **Deployment**: The generated output, contained within this repository, can be deployed to any standard web server (e.g., Apache, Nginx, IIS) or simply accessed locally.
4.  **Runtime (User Side)**: When a user accesses the `index.html` file in their web browser, the browser loads the main HTML structure. It then fetches and applies the generated CSS for styling and executes the complex JavaScript runtime (provided by Storyline). This JavaScript dynamically renders the course slides, manages interactive elements, handles multimedia playback, and tracks user progress or interactions.

## ⚙️ Installation / Quick Start

To view or run the `AB_safety_190220_eng` e-learning module locally, follow these simple steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/iv150320/AB_safety_190220_eng.git
    ```

2.  **Navigate into the project directory**:
    ```bash
    cd AB_safety_190220_eng
    ```

3.  **Open in your web browser**:
    *   The simplest way is to directly open the `index.html` file in your preferred web browser. Locate the `index.html` file in the cloned directory and double-click it, or drag and drop it into a browser window.

    *   For a more robust local serving experience (especially if you encounter issues with local file access or JavaScript security restrictions), you can use a local web server:
        *   **Using `npx serve` (Node.js required)**:
            ```bash
            npx serve .
            ```
            Then, open your browser and navigate to the address provided in the terminal (e.g., `http://localhost:5000`).
        *   **Using VS Code Live Server extension**: If you use Visual Studio Code, install the "Live Server" extension. Right-click on `index.html` and select "Open with Live Server".

This will launch the interactive safety e-learning course, ready for exploration!
